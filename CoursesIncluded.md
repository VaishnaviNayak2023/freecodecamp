## Courses Included

This repository currently includes solutions and notes from the following FreeCodeCamp courses:

1. **Responsive Web Design** – HTML, CSS, Flexbox, Grid, accessibility, and website projects.
2. **JavaScript Algorithms and Data Structures** – Core JavaScript, ES6, regular expressions, debugging, algorithms, and data structures challenges.
3. **Front End Development Libraries** – React, Redux, Bootstrap, jQuery, and project implementations.
4. **Data Visualization** – D3.js, Chart.js, and interactive data visualizations.
5. **APIs and Microservices** – Node.js, Express.js, RESTful APIs, and MongoDB projects.
6. **Quality Assurance** – Chai assertions, testing with Mocha, and practical QA exercises.
7. **Scientific Computing with Python** – Python basics, NumPy, and small projects (if applicable).
8. **Information Security and Quality Assurance** – Security fundamentals, testing, and API security.

## What’s Inside

* **Challenge Solutions** – Step-by-step code examples for exercises in each course.
* **Projects** – Full project implementations with clean code and comments.
* **Video Tutorial Notes** – Notes and code examples from FreeCodeCamp YouTube tutorials.
* **Reference & Learning Material** – Tips, tricks, and explanations to help understand concepts.
